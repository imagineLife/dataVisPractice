A Canvas With Paths

Canvases can render paths.

Paths are...

  • a list of points
  • points connected by segments of lines

Creating a path means...

  • creating a path
  • leverage "drawing commands" to draw the path
  • add styling to the path

Functions to use to draw paths

  • beginPath(): create a path
  • closePath(): Adds a straight line to a path
  • stroke()
  • fill()
